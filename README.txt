Requires python-sfml

Taxi-metric (aka, Manhattan) voronoi diagram geometry generator, with animated demo (run test.py)

keys:
	escape:		exit
	space:		pause
	tab:		reverse
	up/down:	speed

	toggle drawing of:
	c:		 centroids
	n:		 centroids-centroid 'neighbor' linkage
	p:		 centroid polies
