Requires python-sfml

Taxi-metric (aka, Manhattan) voronoi diagram geometry generator, with animated demo (run test.py)

keys:
	escape:		exit
	space:		pause
	tab:		reverse
	up/down:	speed
	c:			toggle drawing of centroids
	n:			toggle drawing of centroids-centroid 'neighbor' linkage
	p:			toggle drawing of centroid polies
